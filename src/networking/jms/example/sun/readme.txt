  Simple JMS program from Sun using ActiveMQ and JNDI
  Refer to resources/jndi.properties
  Broker is created with the same jvm
  Persistent queue is used. 
  The data is stored under the activemq-data directory which is found
  under the project directory root (c:/codejam) 
  
  How to run
  -----------
  Run the SimpleQueueSender program
  Then run the SimpleQueueReceiver program
